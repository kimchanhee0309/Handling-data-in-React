# 1. React로 데이터 다루기

<details>
  <summary>(1) 데이터 주도 UI란?</summary>

  * **한 줄 정의**: **데이터가 바뀌면 -> 화면이 저절로 바뀐다!**
  * **❌ 바닐라 JS - 직접 DOM을 만지는 방식**
    ```JavaScript
    const users = ['홍길동', '김철수', '이영희'];

    // 화면에 그리기
    const ul = document.createElement('ul');
    users.forEach(user => {
      const li = document.createElement('li');
      li.textContent - user;
      ul.appendChild(li);
    });
    document.body.appendChild(ul);

    // 사용자 한 명 추가하면?
    users.push('박민수');
    // 화면은 그대로! DOM을 또 조작해야 함
    ```

  * **✅ React - 데이터만 바꾸면 끝**
    ```JSX
    function UserList() {
      const [users, setUsers] = useState(['홍길동', '김철수', '이영희']);

      return (
        <ul>
          {users.map((user, i) => <li key={i}>{user}</li>))
        </ul>
      );
    }

    // 사용자 추가?
    setUsers([...users, '박민수']);
    // React가 알아서 화면 업데이트!
    ```
</details>

<details>
  <summary>(2) 선언형 vs 명령형(또는 절차형)</summary>

  ```JSX
  // 명령형: 한 단계씩 지시
  const list = document.getElementById('list');
  list.innerHTML = '';
  users.forEach(user => {
    const li = document.createElement('li');
    li.textContent = user;
    list.appendChild(li);
  });

  // 선언형: 결과만 선언
  <ul>
    {users.map(user => <li key={user.id}>{user.name}</li>)}
  </ul>
  ```
</details>

<details>
  <summary>(3) 데이터와 State</summary>

  * **정적 데이터 - 안 변하는 것**
    * 일반 변수로 선언하면 됨
      ```JSX
      function ProductList() {
        const categories = ['전자기기', '의류', '도서', '식품'];

        return (
          <ul>
            {categories.map((c, i) => <li key={i}>{c}</li>)}
          </ul>
        );
      }
      ```
       
  * **동적 데이터 - 변하는 것 -> State로!**
    * 사용자 입력, 버튼 클릭, API 응답 등으로 바뀌는 데이터는 반드시 `useState`로 관리
      ```JSX
      function TodoList() {
        const [todos, setTodos] = useState(['운동하기', '책 읽기']);

        function addTodo() {
          setTodos([...todos, '새 할일']);
        }

        return (
          <div>
            <ul>
              {todos.map((todo, i) => <li key={i}>{todo}</li>)}
            </ul>
            <button onClick={addTodo}>추가</button>
          </div>
        );
      }
      ``` 
</details>

<details>
  <summary>(4) 데이터 구조 설계하기</summary>

  * **Level 1. 단순 배열**
    * 간단한 목록에 적합
      ```JSX
      const fruits = ['사과, '바나나', '오렌지'];
      ```
      
  * **Level 2. 객체 배열⭐**
    * 항목마다 여러 정보를 담을 때
      ```JSX
      const products = [
        { id: 1, name: '노트북', price: 1200000, stock: 5 },
        { id: 2, name: '마우스', price: 30000, stock: 15 },
        { id: 3, name: '키보드', price: 80000, stock: 8 },
      ];
      ```
      * **id라는 고유 번호**가 있어야 특정 항목을 정확히 집어낼 수 있음
      * `map()`의 `key`값으로도 쓰임
      
  * **Level 3. 중첩 구조**
    * 복잡한 데이터를 표현할 때
      ```JSX
      const users = [
        {
          id: 1,
          name: '홍길동',
          profile: {
            age: 25,
            job: '개발자',
            skills: ['React', 'Node.js'],
          };
        };
      ];
      ```
</details>

<details>
  <summary>(5) 핵심 3줄 요약</summary>

  * 1. **React = 데이터 주도 UI**: 데이터만 바꿔라, 화면은 React가 그려준다.
  * 2. **변하는 데이터는 무조건 State**: `useState` 안 쓰면 화면 안 바뀜
  * 3. **실무에선 객체 배열 + id**: 거의 모든 상황에서 이 패턴을 씀
</details>

# 3. filter로 아이템 삭제하기

<details>
  <summary>(1) filter 메서드 복습</summary>

  * **기본 사용법**
    ```JSX
    const numbers = [1, 2, 3, 4, 5, 6];

    // 짝수만 통과
    const evenNumbers = numbers.filter(num =>
      num % 2 === 0);
    console.log(evenNumbers);  // [2, 4, 6]

    // 3보다 큰 수만 통과
    const greaterThanThree = numbers.filter(num => num > 3);
    console.log(greaterThanThree);  // [4, 5, 6]
    ```
    
  * **객체 배열 필터링**
    ```JSX
    const users = [
      { id: 1, name: '홍길동', active: true },
      { id: 2, name: '김철수', active: false },
      { id: 3, name: '이영희', active: true },
    ];

    // 활성 사용자만
    const activeUsers = users.filter(user => user.active);
    // -> 홍길동, 이영희

    // id가 2가 아닌 사용자만
    const withoutId2 = users.filter(user => user.id !== 2);
    // -> 홍길동, 이영희(김철수 제외)
    ```
</details>

<details>
  <summary>(2) 삭제 기능 구현</summary>

  ```JSX
  function TodoList() {
    const [todos, setTodos] = useState([
      { id: 1, text: '운동하기' },
      { id: 2, text: '책 읽기' },
      { id: 3, text: '코딩하기' },
    ]);

    function deleteTodo(id) {
      // "클릭한 id가 아닌 것만 남겨라"
      const filtered = todos.filter(todo => todo.id !== id);
      setTodos(filtered);
    }

    return (
      <ul>
        {todos.map(todo => (
          <li key={todo.id}>
            {todo.text}
            <button onClick={() => deleteTodo(todo.id)}>삭제</button>
          </li>
        ))}
      </ul>
    );
  }
  ```
</details>

<details>
  <summary>(4) 조건부 삭제(일괄 삭제)</summary>

  ```JSX
  function TodoList() {
    const [todos, setTodos] = useState([
      { id: 1, text: '운동하기', done: true },
      { id: 2, text: '책 읽기', done: false },
      { id: 3, text: '코딩하기', done: true },
    ]);

    function deleteCompleted() {
      // 완료되지 않은 애들만 남겨라
      setTodos(todos.filter(todo => !todo.done));
    }

    return (
      <div>
        <button onClick={deleteCompleted}>완료된 항목 삭제</button>
        <ul>
          {todos.map(todo => (
            <li key={todo.id}>
              {todo.text} {todo.done && 'v'}
            </li>
          ))}
        </ul>
      </div>
    );
  }
  ```

<details>
  <summary>(5) ⭐필터링 UI - 렌더링 시점 필터링</summary>

  ```JSX
  function ProductList() {
    const [products, setProducts] = useState([
      { id: 1, name: '노트북', price: 1200000, category: '전자기기' },
      { id: 2, name: '마우스', price: 30000, category: '전자기기' },
      { id: 3, name: '티셔츠', price: 25000, category: '의류' },
    ]);
    const [filter, setFilter] = useState('all');

    // 렌더링될 때마다 필터링된 배열 생성
    const filteredProducts = products.filter(product => {
      if (filter === 'all') return true;
      if (filter === 'electronics') return product.category === '전자기기';
      if (filter === 'clothes') return product.category === '의류';
      return true;
    });

    return (
      <div>
        <div>
          <button onClick={() => setFilter('all')}>전체</button>
          <button onClick={() => setFilter('eletronics')}>전자기기</button>
          <button onClick={() => setFilter('clothes')}>의류</button>
        </div>

        {filteredProducts.map(product => (
          <div key={product.id}>{product.name}</div>
        ))}
      </div>
    );
  }
  ```
